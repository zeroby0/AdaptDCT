AdaptDCT: A Deep Learning-Guided Multi-Scale DCT Compression Framework for Edge Devices

This paper has been submitted to SIGCOMM 2025 for review. This webpage provides anonymous access to the project's source code for reviewers.

This repository contains the complete implementation of AdaptDCT, including:

- Core compression algorithm implementation
- Deep learning models and training scripts
- Evaluation framework and benchmarking tools

The code is provided as-is and has been anonymized to maintain the double-blind review process. All identifying information has been removed from comments and documentation.
